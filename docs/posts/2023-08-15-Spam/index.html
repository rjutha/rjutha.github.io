Creating a Shiny UI in a Quarto document

For this week’s Tidy Tuesday I to experiment with integrating shiny widgets in Quarto files. I took inspiration from Albert Rapp’s article possibilities the possibilities.

Using Quarto as an alternative to R Markdown

I’m not aware of all the major differences between Quarto and R Markdown but I’ve been experimenting with Quarto more recently, so much so that I created this blog. I’m enjoying the aesthetic and customization of Quarto and plan to incorporate it into my workflow. Noteworthy features for me so far are table of contents, collapsible code blocks, and embedding tools make for a clean UI. It’s also interesting that Quarto can be used for projects coded in other languages like Python further expanding its versatility.

Hosting on shinyapps.io

Since the website contains interactive shiny widgets, they cannot be rendered on github pages. Because of this I hosted the app on shinyapps.io and embedded the link below. The app can be viewed via this [link](https://sway.shinyapps.io/Spam/.

Final Remarks

Combining Quarto and Shiny together is relatively simple and provides a great way to improve the overall look and practicality.
